Next-Generation Sequencing Data is Compositional

Next-generation sequencing measurements provide only relative abundances of transcripts (Robinson and Smyth 2007; Anders and Huber 2010; Robinson and Oshlack 2010; Law et al. 2014; Lovell et al. 2015). The relative nature of next-generation sequencing has important consequences for downstream analyses.

  • There are several reasons NGS data are relative.
    • Finite sample volume from which DNA/RNA is extracted
    • Finite number of reads available on each platform (as the number of reads increases for 1 transcript the number of available reads for the other transcripts decreases, Fig. 1)
    • Read depth (number of reads allocated to a sample) varies from run-to-run and sample-to-sample
  • Traditional methods can produce misleading results for compositional data
    • Spurious correlation (Pearson...)
    • Univariate results may be misleading (Lovell et al. 2015)
    • Negative bias of the covariance matrix

Properties of Compositional Data

Relative abundance data arise from an arbitrary sum constraint, \(\sum_{i=1}^{D} t_i = \mathbb{T} \), where \(t_i\) gives the number of reads to sample \(i\) in a sequencing run with \(D\) samples.

  1. The sum contraint, \(\mathbb{T}\), can change between platforms, sequencing runs, and samples.
  2. This leads to an arbitrary difference in scale between samples which various normalization methods have sought to correct.

Figure 1: This animation shows the impact of changes to a single transcript on the relative abundance of the other transcripts being measured. Ignoring the constraints imposed by compositional data can lead to spurious results.

Compositions lie in the \(D-1\) dimensional simplex (\(S^{D-1}\)). As a result of the constraints on compositional data, the covariance matrix of an un-transformed composition is difficult to interpret.

Compositional Operations and their Properties

Many problems induced by compositional data can be aleviated by working with log-ratios of components. John Aitchison (1985) outlined several operations for working with compositions:

  • Additive Log-Ratio (ALR): \( ALR (x_i) = log (\frac{x_i}{x_D}) \), where \(x_D\) is the \(D^{th}\) component of the vector X.
    • Interpretable variance-covariance matrix
    • Does not treat all components equally as one component is used for denominator
  • Centered Log-Ratio (CLR): \( CLR (x_i) = log( \frac{x_i}{g(X)}) \), where \(g(X)\) is the geometric mean of X.
    • Interpretable variance-covariance matrix
    • Preserves the dimension of the original composition
    • Resulting variance-covariance matrix is singular

Figure 3.The ALR transformation reduces the dimension of the composition but preserves the absolute relationships between the components. Transformed data can then be used with standard multivariate methods.

Counts per Million: A compositional operation

Counts per million (CPM) transformation is a simple, and widely used, standardization method which adjusts reads of individual transcripts by the total number of reads allocated to the sample. Generally, values are then log-transformed. $$log_2 (CPM(x_i)) = log_2(\frac{x_i+0.5}{(\sum_{j=1}^D x_j)+1} \times 10^6)$$ where, \(x_i\) is the read count for gene i in a sample with D genes measured.

  • What does CPM accomplish?
    • CPM simply rescales the data, a compositional closure operation.
    • Reads from each sample then sum to \(\approx\)1 million
  • What does CPM not accomplish?
    • Data still resides in the compositional simplex
    • Traditional methods may still produce spurious results
      • Spurious correlation
      • Negatively biased covariance matrix
      • etc.

CPM alleviates some, but not all, of the challenges imposed by compositional data.

Sub-compositions and Amalgamations

Recognizing the compositional nature of sequencing data leads to several useful properties:

  • Sub-Compositions
    • Ability to isolate sub-groups of probes from the sequenced gene set for focused analysis
    • Can reduce noise from excluded genes with high variation
  • Amalgamations
    • Can collapse groups of genes with the same functionality into a single proportion
    • Facilitates the use of exploratory techniques for finding differences in pathways/functional groups
    • Amalgamations retain compositional structure

Composition-Based Quality Control

We capitalize on the compositional nature of RNA-seq data to develop a quality control metric for sequencing runs

  • The number of reads/sample is constrained by total reads/sequencing run.
  • We expect the reads to be proportionally allocated accross samples. Samples with unusully low or high reads indicate a possible problem with the sample preparation or sequencing run
  • CLR transformation allows the use of standard outlier detection methods (Cook's Distance/ GOF).

Batch Effect Detection (without normalization)

Batch effects are technical variation that can possibly confound biologic variation

  • Typical methods for detection of batch effects:
    • Multivariate methods - Principle Components Analysis or Hierarchical Clustering
    • Visual inspection
  • Traditional batch effect detection with multivariate methods requires prior sample normalization
    • Requires multiple samples and must be re-normalized when a new sample is added
    • Generally assumes most transcripts are equally expressed
    • Euclidean distance can be misleading when measuring distances between compositions (Aitchison 1985)
  • CLR transformation permits batch effect detection using PCA without additional normalization
    • CLR covariances are interpretable in \( R^k \)
    • Aitchison distance (1986) provides compositional equivalent to Euclidean distance
      • \[ d_A(x, y) = ||clr(x) - clr(y)||_2 \]
    • Enables the use of methods which utilize correlations or distances (e.g. PCA, clustering, outlier detection)
    • Applied to individual samples - no need to re-transform if new samples are added
    • No assumption about the similarity of samples (e.g. most transcripts are equally expressed)

Sub-Compositions

  • Ability to isolate sub-groups of probes from the sequenced gene set for focused analysis
  • Can reduce noise from excluded genes with high variation

The sub-compositional consitency of compositional methods can allow the evaluation of control or housekeeping probes for the evaluation of sample quality

Amalgamations

  • Can collapse groups of genes with the same functionality into a single proportion
  • Facilitates the use of exploratory techniques for finding differences in pathways/functional groups by reducing the dimension of the data
  • Amalgamations retain compositional structure
  • Examples of hierarchical amalgamations:
    • Total reads over entire run
    • Sample level total reads
    • Reads assigned to functional groups of transcripts (Housekeeping, oncogenes, etc.)

HTG Molecular Immuno-Ocology (IO) consists of 549 probes which can be amalgamated into 10 functional groups. This dimensional reduction allows the use of visual exploratory methods such as PCA or clustering (see example analysis).
